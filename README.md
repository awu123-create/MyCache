# MyCache
该项目实现了一个轻量级的分布式缓存系统，参考了 groupcache 的核心设计思想，并在此基础上构建了本地缓存、节点间通信、一致性哈希负载分发以及请求合并等关键能力。系统提供可扩展的缓存分组管理、可插拔的数据源加载回调以及简洁的 HTTP 通信协议，适用于高并发场景下的数据缓存需求。  
本项目的主要特性包括：  
* 本地 LRU 缓存管理：每个节点维护独立的LRU缓存并提供并发安全访问。  
* 一致性哈希节点选择：通过一致性哈希算法为每个key选择固定节点，支持节点动态上下线。  
* 分布式数据加载：当本地节点无缓存时，可根据一致性哈希从远程节点拉取数据。  
* 用户自定义数据源回调：支持用户为缓存miss定义数据加载函数，从数据库或其他持久化存储读取。  
* 请求合并（Singleflight）：同一个key在并发访问时仅触发一次数据加载，避免缓存击穿。  
* 轻量级 HTTP 通信接口：节点间通过HTTP + protobuf完成高效交互。  
源代码在master分支。
